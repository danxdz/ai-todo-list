# Todo List App

A beautiful and feature-rich todo list application built with React and Tailwind CSS, created by AI via Telegram.

## Features

### ✨ Core Functionality
- **Add Todos**: Create new todo items with a simple input field
- **Mark Complete**: Toggle todo completion status with interactive checkboxes
- **Edit Todos**: In-place editing with keyboard shortcuts (Enter to save, Escape to cancel)
- **Delete Todos**: Remove individual todos with a single click
- **Bulk Actions**: Mark all todos as complete or clear all completed todos

### 🎯 Advanced Features
- **Filtering**: View all, active, or completed todos
- **Statistics Dashboard**: Real-time counts of total, active, and completed todos
- **Persistent Storage**: Todos are automatically saved to localStorage
- **Responsive Design**: Works perfectly on desktop and mobile devices
- **Beautiful UI**: Modern gradient backgrounds, smooth animations, and intuitive interactions

### 🎨 User Experience
- **Empty States**: Helpful messages when no todos exist or filters return no results
- **Visual Feedback**: Completed todos are crossed out and styled differently
- **Keyboard Support**: Press Enter to add todos or save edits
- **Hover Effects**: Interactive elements provide visual feedback
- **Color-coded Statistics**: Different colors for total, active, and completed counts

## Technology Stack

- **React 18.2.0** - Modern React with hooks
- **Tailwind CSS 3.3.0** - Utility-first CSS framework
- **LocalStorage** - Client-side data persistence
- **Create React App** - Development toolchain

## Getting Started

### Prerequisites
- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/danxdz/todo-list-5153324774-1757368888878.git
cd todo-list-5153324774-1757368888878
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Available Scripts

- `npm start` - Runs the app in development mode
- `npm run build` - Builds the app for production
- `npm test` - Launches the test runner
- `npm run eject` - Ejects from Create React App (one-way operation)

## Usage

1. **Adding Todos**: Type your todo in the input field and press Enter or click "Add Todo"
2. **Completing Todos**: Click the checkbox next to any todo to mark it as complete
3. **Editing Todos**: Click the "Edit" button, modify the text, and press Enter to save
4. **Deleting Todos**: Click the "Delete" button to remove a todo permanently
5. **Filtering**: Use the filter buttons to view all, active, or completed todos
6. **Bulk Actions**: Use "Mark All Complete" or "Clear Completed" for batch operations

## Data Persistence

All todos are automatically saved to your browser's localStorage, so your todos will persist between browser sessions. No backend server required!
